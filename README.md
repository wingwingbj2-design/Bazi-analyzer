# 兒童八字教育分析系統

一個基於傳統八字學與現代教育心理學的兒童教育分析工具，幫助家長了解孩子的性格特質、潛能發展方向，並提供分齡具體教育建議。

## 功能特色

- **自動八字排盤**：輸入兒童出生資料，系統自動計算八字
- **性格特質分析**：基於八字分析兒童性格特點
- **潛能發展方向**：提供個性化的潛能發展建議
- **分齡教育建議**：針對不同年齡段提供具體教育方法
- **安全保密**：不儲存用戶資料，保護隱私
- **響應式設計**：支持手機、平板、電腦訪問

## 快速開始
### 本地運行

1. 克隆項目
bash
git clone https://github.com/WING-AI-2024/bazi-analyzer.git
cd bazi-analyzer
2. 安裝依賴
bash
pip install -r requirements.txt
3. 啟動服務
bash
python server.py
4. 訪問應用

http://localhost:5000/
### Railway 部署

1. 訪問 [Railway.app](https://railway.app)
2. 點擊 "New Project" → "Deploy from GitHub repo"
3. 選擇此倉庫
4. Railway 會自動構建和部署

## 安全與隱私

- **不儲存數據**：所有分析結果僅在當前會話中顯示
- **無需註冊**：直接使用，無需創建賬戶
- **本地計算**：核心分析邏輯在服務器端完成
- **安全傳輸**：使用HTTPS加密傳輸
